前端开发知识点

2019-06-21 /
  • JavaScript
  • Node.js
  • Web开发
  • jQuery
  • 前端开发
  • 前端框架
 / 点击

公司要做定岗定级，需要前端知识点用于评价，梳理了以下类型知识点供参考。
公司主要做Java Web项目开发，Node.js知识也部分使用，因此梳理以下类型知识点：JavaScript、jQuery、前端框架、Node.js。

JavaScript

了解

能读懂简单的JavaScript代码

1、掌握基本语法（如：变量、语句、方法、判断、循环、调用）；
2、掌握基本运算符，掌握基本比较符；
3、掌握基本事件类型（如：onclick、onchange、onmouseover、onkeyup、onblur、onload等）；
4、掌握一种获取DOM元素的方法（getElementById、Jquery选择器等）。

掌握

能开发完整的JavaScript代码

1、掌握数据类型（如：字符串、数字、布尔值、对象、数组、Undefined和Null）；
2、掌握基础对象（如：String、Object、Array等）及其常见接口；
3、了解作用域全局作用域（window/global），了解局部作用域（方法作用域）；
4、掌握两种事件绑定/监听的方式（addEventListener、（IE7、8）attachEvent、el.onclick = function(){}、jQuery事件绑定等），了解事件解绑的方式（removeEventListener、（IE7、8）attachEvent、el.onclick = null、jQuery事件解绑等）；
5、了解DOM元素的属性（id、name、value、innerHTML等）；
6、了解事件Event对象的属性（type、currentTarget、target、keyCode等）。

熟练

了解JavaScirpt的开发技巧

1、掌握Ajax开发技术；
2、了解事件冒泡、事件委托；
3、掌握JSON和对象的相互转换；
4、了解值传递和引用传递，了解浅复制和深复制；
5、掌握窗口中数据传递和交互的技巧；
6、了解Cookie技术，掌握cookie的操作技巧；
7、掌握异步编程方法（基于回调，常在事件中、Ajax中使用）；
8、了解一种常用的开发工具库（jQuery、Underscore、Lodash等），了解一种常用的开发框架（Vue、React、Angular、BackBone等）；
9、项目开发中，具有封装工具方法的经验；
10、正则表达式的用法，正则处理操作。

擅长

了解JavaScript的相关原理

1、了解ECMAScript的版本（ES3、ES5、ES6），了解ES6新特性；
2、了解变量穿透块级作用域的问题（var存在，let解决）；
3、了解浏览器的基本实现原理；
4、了解HTTP协议，了解HHTP请求（请求行、消息报头、请求正文），了解HTTP响应（状态行、消息报头、响应正文）；
5、理解原型链；
6、掌握闭包的实现；
7、了解模块化编程实现方案；
8、项目开发中，具有解决兼容性问题的经验；
9、了解一种JS中常见的安全漏洞，掌握修复方案。

jQuery

分档的标准类似JavaScript分档的标准

了解

1、掌握jQuery基本选择器（id选择器、tagName选择器、className选择器）；
2、掌握一种jQuery绑定事件方法（on、bind、delegate/live、click等简写事件）；
3、了解jQuery的链式调用。

掌握

1、掌握一种jQuery复杂选择器（后代选择器、群组选择器、属性选择器）；
2、了解元素内容的方法（html、text、val）、了解操作元素属性的方法（attr、prop）、了解操作元素样式的方法（css、addClass、width、height等）；
3、了解操作节点的方法（创建$()、插入append、删除remove、清空empty）；
4、了解jQuery事件对象的属性（type、target、currentTarget、pageX/clientX）；
5、掌握元素遍历的方式（each），以及跳出遍历的语句（return false）；
6、掌握jQuery实现Ajax的方法（ajax、get、post）；

熟练

1、掌握jQuery过滤器（基本过滤器、内容过滤器、可见性过滤器、子元素过滤器等，如”:first”、”:not()”、”:eq()”,”:hidden”,”:first-child”）；
2、掌握jQuery事件委托的实现方式，了解事件命名空间的作用，掌握如何模拟事件触发（trigger、click等简写触发）；
3、事件响应中阻止冒泡的方式（event.stopPropagation()、return false），以及两种方式的区别；
4、掌握jQuery实现Ajax的进阶技巧，了解同步异步，了解请求事件响应（success、error、complete），了解响应数据类型dataType（html、script、json、text等）；
5、掌握jQuery实现交互动画效果的方法（显隐show/hide、滑动slide、淡入淡出fadeIn/fadeOut、自定义动画animate）；
6、使用过一款jQuery插件（layer、select2等），使用过一款基于jQuery的UI组件库（jQueryUI、easyUI、layUI等）。

擅长

1、了解jQuery无new构造原理；
2、了解$(function () {})和onload的区别；
3、了解Ajax处理跨域的方式（代理、XHR2、jsonP）；
4、了解jQuery对AMD规范的支持，了解支持AMD规范/CMD规范的文件加载工具（Require.js/Sea.js）；
5、拥有开发jQuery插件的经验。

前端框架

分档的标准类似JavaScript分档的标准；评价时依据参评人员熟悉的框架具体展开

了解

1、了解过一款前端框架（Vue、React、Angular、Rxvm等），了解基本开发方式。

掌握

1、掌握组件的实现方式；
2、掌握数据更新的方法（对象、数组）；
3、掌握视图的开发方式，了解视图中的插值写法；
4、了解视图中绑定事件的指令（Vue:”v-on”，React：类似原生(转驼峰)”onClick”，Angular：”ng-“，Rxvm：”on-“）；
5、了解视图中实现数据双向绑定的指令（Vue：”v-model”，Angular：”[(ngModel)]”，Rxvm：”model”，React：无，通过value和onChange手动绑定）。

熟练

1、了解组件生命周期钩子（事件）。
2、掌握监听数据变化的实现方式（如：Vue：watch，React:props+componentWillReceiveProps/state+componentWillUpdate，Angular：$watch，Rxvm：watch）；
3、掌握全局组件的封装。

擅长

1、项目搭建时，了解选择前端（MVVM）框架的原因（大规模、易维护），了解选择某款框架及其周边的思路（灵活性、语言、学习成本、兼容性）；
2、了解单向数据流原理；
3、了解双向数据绑定实现原理；
4、了解状态管理（内部实现：如Rxvm实现双向数据流间接支持，工具库支持：如Vuex、Redux、ngrx/store等）；
5、了解路由管理，了解单页应用与多页应用的优劣对比。

Node.js

了解

1、了解Node.js运行环境的部署；
2、了解Node.js基本语法规则。

掌握

1、了解npm包管理器，会使用npm命令安装需要的包，了解本地安装和全局安装；
2、掌握 require 语句载入模块/包；
3、了解Node.js回调函数，可列举一个阻塞代码案例和一个非阻塞代码案例；
4、了解Node.js内置的模块（os、path、http、fs等）。

熟练

1、了解EventEmitter类，了解绑定事件和触发事件的方法，了解EventEmitter类常用的事件（newListener、removeListener、error）。
2、掌握创建服务器的方法，了解接受请求和响应请求；
3、了解Node.js对数据库（mysql、MongoDB等）的连接、CRUD操作；
4、了解一款Node.js Web开发框架（Express、Koa等）。

擅长

1、了解Node.js运行原理，深入理解Node.js核心特性（单线程、异步IO、事件驱动）；
2、了解Node.js项目中的使用场景，能在合适的场景下选择Node.js或其他语言及框架；
3、了解Package.json，了解Package.json主要属性及其作用；
4、掌握Webpack或Gulp/Grunt等前端打包工具（理解工具打包原理），掌握通过配置文件来使用工具，掌握（结合前端框架）快速搭建前端项目。
